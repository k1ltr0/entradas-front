## Quick Start

### For Developers

```bash
# Already installed in dashboard-react
cd dashboard-react
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities react-icons --legacy-peer-deps
```

### For Users

1. Navigate to `/site-builder` in the dashboard
2. Select a template
3. Edit sections
4. Save and publish
